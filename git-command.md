# What is Branch 
## To check current branch
```
git branch 
```
## To create a New Branch 
```
git checkout -b devtest

```
## To go back previous branch 
```
git checkout master
```
## to delete branch 
```
git branch -d branchname
```
## to force delete branch 
```
git branch -D branchname
```
## to Delete branch from remote 
```
git push origin --delete branchname
```
# Commit

